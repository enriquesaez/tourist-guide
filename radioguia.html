Radioguide

The portable Wireless Audio Transceiver system consists of an audio transmitter and a set of receivers, When using this system, a tour guide can introduce scenic sites to his/her touring party by wireless voice, without interfere other touring groups.

What is it?

A Radioguide system is a set of receivers that pick up the voice, emitted by a transmitter by frequency, within a specific radio-frequency wave (RF).

It is the same system as a traditional radio (FM) that emits by modulated frequency but working at a higher frequency; as an analogue UHF television channel (Ultra High Frequencies).

The radio guides can be used in all cases in which one or more people need to follow a set of explanations live, by a guide, with the freedom to move or not to follow the look without losing that explanation.
